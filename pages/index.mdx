# wallet.page

Test whether your browser wallet supports common [EIPs](https://eips.ethereum.org/) and JSON-RPC methods — similar in spirit to the [MetaMask test dapp](https://metamask.github.io/test-dapp/), but organized as documentation with an interactive panel on each page.

## How it works

1. Start with [Connect](/connect) if you have multiple extensions or want to review discovery UX.
2. On any demo page, choose a wallet from the picker — the session is shared site-wide.
3. Run the action and read the result — success, user approval, or a signal about what to implement next.

All requests stay in your browser.
