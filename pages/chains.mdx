# Chains

Networks are identified by `chainId` (hex). Wallets expose read and write RPCs so dapps can read the active chain or ask the user to add or switch networks.

<ChainQuickGrid />

## eth_chainId

Returns the wallet’s current chain. No prompt when already connected.

**RPC:** `eth_chainId` → hex string (e.g. `0x1`)

<EthChainIdDemo />

## wallet_switchEthereumChain (EIP-3326)

Changes the active chain when the network is already known to the wallet.

**RPC:** `wallet_switchEthereumChain({ chainId })`

:::note
Error **4902** means the chain is not in the wallet yet — use `wallet_addEthereumChain`, then switch again.
:::

<SwitchChainMiniDemo />

## wallet_addEthereumChain (EIP-3085)

Registers chain metadata (RPC URL, currency, name) so the user can switch later.

**RPC:** `wallet_addEthereumChain({ chainId, chainName, rpcUrls, nativeCurrency, ... })`
