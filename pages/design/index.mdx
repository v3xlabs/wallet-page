# Design

Welcome to the wallet UI focused side of this documentation.
This section aims to run you through common wallet experiences rather than rpc calls.
It outlines a plethora of things from onboarding, accounts, all the way to swapping, displaying assets, and supporting different locales.

Note that some of the demo's in this section contain fictive numbers and don't perform any actions onchain.
They are for illustrative & educational purposes.
