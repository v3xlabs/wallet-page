# Watch asset (EIP-747)

[EIP-747](https://eips.ethereum.org/EIPS/eip-747) (`wallet_watchAsset`) lets a dapp ask the wallet to display a token the user may not have imported yet.

**RPC:** `wallet_watchAsset({ type: 'ERC20', options: { address, symbol, decimals, image } })`
